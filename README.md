[ For Roblox Platform ]

A simple ModuleScript that creates arrow beams that point players to their objectives.

**_Note: Make sure your beam parented in ModuleScript!_**

Documentation: 

Arrow.pathToPart(character, part)
-> Clones arrow beam, parents the beam to the player's character and sets the Attachment to the character and the part.

Arrow.destroyPathToPart(character, part)
-> Destroys the arrow beam.
